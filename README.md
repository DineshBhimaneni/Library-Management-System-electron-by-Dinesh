Library Management System

A desktop-based Library Management System (LMS) built using Electron. This application helps manage books, students, staff, loans, and events efficiently in a library environment.

Features

Book Management: Add, update, and delete books from the catalog.

Student & Staff Management: Maintain records of students and staff.

Loan Management: Track issued and returned books.

Fines & Payments: Calculate and manage fines for late returns.

Reports: Generate detailed reports for books, loans, and events.

Search Functionality: Quickly search for books or users.

User-Friendly Interface: Intuitive HTML/CSS/JS frontend powered by Electron.

Installation

Clone the repository

git clone https://github.com/DineshBhimaneni/Library-Management-System-electron-by-Dinesh.git
cd Library-Management-System-electron-by-Dinesh/electron


Install dependencies

npm install


Run the application

npm start

Usage

Open the app via Electron.

Navigate through the menus to manage books, students, staff, loans, fines, and reports.

Use the search bar to quickly find books or users.
